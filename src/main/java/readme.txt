Program written by Josh DeWitt for CSCI4500 in the Summer 2013 session.

Build the program using the makefile.
    $ make

Run the program using the main class.
    $ java main.DeadlockDetector

Program reads from standard input which can be redirected as normal.
    $ java main.DeadlockDetector < test_input

Turn on verbose mode by adding any parameter to the command line.
    $ java main.DeadlockDetector -v
    $ java main.DeadlockDetector anything
